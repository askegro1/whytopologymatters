# Project Overview

Each figure is generated by its own MATLAB script and produces publication-ready PDF outputs and corresponding `.fig` files for editing.

---


## What Each Script Produces

| Script | Output Figures / Tables | Output Folder |
|--------|--------------------------|---------------|
| `Fig3_SuppFig1.m` | Figure 3 & Supplementary Figure 1 | `PlotResults_Fig3_SuppFig1/` |
| `Fig4_SuppTableS2.m` | Figure 4 & Supplementary Table S2 (`.tex`) | `PlotResults_Fig4_SuppTableS2/` |
| `Fig5_SuppFig2_SuppFig3.m` | Figure 5, Supplementary Figures 2 & 3 | `PlotResults_Fig5_SuppFig2_SuppFig3/` |

Each of these folders contains its own detailed README file with input requirements and step-by-step instructions.

---

## Software Requirements

| Software | Version |
|----------|---------|
| MATLAB   | R2023b |
| Toolboxes | Statistics & Machine Learning Toolbox |
| OS Compatibility | Windows, macOS, Linux |

---

## How to Run Scripts

General steps (same for each script):

1. Open **MATLAB**
2. Set the working directory to the project folder
3. Make sure the required input files are present (see script-specific README)
4. Run the script, e.g.:

```matlab
Fig3_SuppFig1
Fig4_SuppTableS2
Fig5_SuppFig2_SuppFig3
```

Output files are automatically saved into their respective `PlotResults_*` folders.
